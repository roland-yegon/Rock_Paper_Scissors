# Rock Paper Scissors Game

This is a web-based Rock–Paper–Scissors game built with **HTML**, **CSS**, and **JavaScript**.

## How to Run
Just open `rock_paper_scissors.html` in any browser.

## Features
- Random computer choice
- Player vs computer logic
- Live win/draw/lose counter
- Simple and clean UI
